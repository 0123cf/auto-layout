### USE
![image.png](https://github.com/0123cf/auto-layout/blob/master/imgs/f68h2j9k2z8b4m24k3l.png?raw=true)

* 第一个是在插入布局。
* 第二个是控制区，可以选中要修改的位置、删除当前位置的布局、新增兄弟节点等
* 第三个是编辑属性，可以修改长度、添加文字、修改颜色、添加图片
* 第四个是源码预览，代码下载等功能，点击右下角
